ScreenPlus HD Repository para Kodi

Bem-vindo ao ScreenPlus HD Repository!
Este repositório permite instalar addons para Kodi de forma simples e rápida.

Requisitos

Kodi 19 ou superior (Matrix, Nexus, etc.)

Conexão à Internet

Instalação pelo URL

Abra o Kodi.

Clique em Configurações (ícone de engrenagem).

Vá para Gestor de ficheiros → Adicionar fonte.

Clique em <Nenhum> e insira o seguinte URL:

https://screenplushd.github.io/repository.screenplus/


Nomeie a fonte como ScreenPlus HD (ou outro nome à sua escolha) e clique em OK.

Volte ao menu principal e selecione Add-ons → Instalar a partir de um ficheiro zip.

Clique em ScreenPlus HD → selecione o ficheiro repository.screenplus-x.x.x.zip para instalar o repositório.

Após a instalação, vá para Instalar a partir do repositório → ScreenPlus HD Repository → escolha o addon que deseja instalar.
